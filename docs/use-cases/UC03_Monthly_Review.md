# UC03: 월간 진행 검토 (Monthly Visualization & Review)

## 1. 개요

사용자가 한 달 동안의 루틴 유지 현황을 시각적으로 확인하고 분석하는 과정입니다.

## 2. 관련 요구 사항

- PRD 3.1 (월별 그리드 뷰)
- PRD 3.4 (월간 진행 상황 시각화)

## 3. 액터 (Actor)

- 사용자 (User)

## 4. 기본 흐름 (Success Scenario)

1. 사용자가 메인 대시보드에 접속합니다.
2. 시스템은 자동으로 현재 년/월을 계산하여 해당 월의 그리드를 렌더링합니다.
3. 사용자가 상단의 '월 이동' 버튼(<, >)을 사용하여 다른 달로 이동합니다.
4. 시스템은 이동한 월에 해당하는 저장된 데이터를 불러와 그리드를 다시 그립니다. (슬라이드 전환 애니메이션 적용)
5. 사용자는 화면 상단 또는 측면의 통계 위젯을 통해 다음 정보를 확인합니다:
   - **월간 총 달성률**: 전체 셀 대비 완료된 셀의 비율
   - **습관별 스트릭(Streak)**: 현재 연속으로 달성 중인 일수
6. 사용자가 특정 습관 행(Row)을 보며 한 달간의 유지 패턴을 한눈에 파악합니다.

## 5. 시각적 피드백

- **히트맵 효과**: 달성률이 높은 날은 그리드 헤더의 색상이 더 진하게 표시되는 등 전체적인 성과를 직관적으로 보여줍니다.
